Данный набор скриптов выделяет из ЭЭГ эпох признаки, которые в последствии будут использованы для их классификации. Каждая эпоха (эксперимент)
представляет из себя 3-х мерные массив. Временной отсчет х канал х попытка. По временной шкале 750 отсчетов, что соответсвует промежутку [-0.5,1.5] c.
По временной шкале 23 канала - 'FZ','F3','F4','Cz','C3','C4','PZ','P3','P4','P1','P2','PO7','PO8','PO3','PO4','Oz','O1','O2','POz','FP2','VEOG', 
'HEOGL', 'HEOGR'. Из них использовались 19 - 'FZ','F3','F4','Cz','C3','C4','PZ','P3','P4','P1','P2','PO7','PO8','PO3','PO4','Oz','O1','O2','POz'
Количество попыток завист от эксперимента.
Данные скрипты извлекают два различных набора признаков - амплитудные и вейвлетные.
Получение амплитудных фич.
В начале из всего временного промежутка вычитается среднее по времени в пределах одного канала в промежтуке от 200 до 300 мс. 
Операция производится для всех каналов и попыток. Далее для получения амплитудных фич используется промежуток 200-500мс. 
Для получения признаков по данному промежутку скользит окно длинной 50 мс с шагом в 20мс (Первое окно 200-250мс, последнее 450-500мс). Признаками 
являются усредненные данные внутри каждого окна. Таким образом для каждого канала выделяется 13 признаков. Признаки внутри полученные от различных
каналов конкатенируются внутри каждой попытки. В итоге получаются две двумерные матрицы для целевых и нецелевых попыток соответсвенно. По строкам 
записаны попытки, по столбцам различные признаки.
Получение вейвлетных фич.
Для выделения вейвлетных признаков используется временной промежуток 50-500ms (275-500 отсчеты). Для преборазования выбираются вейвлеты Морле.
Перед преобразованием из временного промежутка вычитается среднее по времени внутри одного канала. Вейвлет преобразование проводилось с масштабами
вейвлетов, соответсвующими частотам от 5 до 30 Гц с шагом в 0.5 Гц. Всего получился 51 масштаб. Таким образом на выхоже получилось два 4-х мерных массива
(масштаб х временной отсчет х канал х попытка).
